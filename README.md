Whether you're using a Logitech, Razer, Microsoft, or any standard USB/Bluetooth mouse, Mos enhances your scrolling experience with customizable settings and native macOS integration.

📖 What is Mos for Mac?

Mos is a free and open-source macOS application that improves mouse scrolling by adding smooth scrolling, independent scroll speed adjustment, acceleration customization, and support for a wide variety of external mice.

✨ Key Features
🖱 Smooth Scrolling

Enjoy fluid, trackpad-like scrolling with supported external mice.

⚙️ Adjustable Scroll Speed

Customize scrolling speed independently from macOS system settings.

🎯 Per-App Settings

Configure different scrolling behaviors for specific applications.

🔄 Scroll Direction Control

Customize natural or traditional scrolling preferences.

🚀 Lightweight Performance

Runs efficiently in the background with minimal system resource usage.

🖥 Wide Mouse Compatibility

Supports most USB and Bluetooth mice.

📊 Menu Bar Access

Quickly access settings and preferences from the macOS menu bar.

🍎 Native macOS Experience

Optimized for both Intel and Apple Silicon Macs.
